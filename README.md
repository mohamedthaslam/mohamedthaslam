I'm a **Full Stack Developer UI/UX Enthusiast** who loves crafting visually stunning, interactive web experiences.

Instead of stopping at templates, I build custom designs from scratch — **glassmorphism UIs, 3D animations with Three.js**, and smooth scroll experiences that leave an impression.
